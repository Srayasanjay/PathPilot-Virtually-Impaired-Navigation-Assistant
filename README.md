# PathPilot-Virtually-Impaired-Navigation-Assistant
A Navigation assistant for visually impaired users, build using various AI models. This project have the following functions:
1. Object detection model using Yolov8
2. Depth perception module using MiDas
3. Text detection using TesseractOCR
4. Scene Description using LLM
5. Memory model using ORB feature extractor and VSLAM
